# demo-e_commerce
(TH) ระบบเว็บไซต์ E-Commerce เต็มรูปแบบ พัฒนาด้วย React.js และ Next.js สำหรับฝั่ง Frontend ออกแบบ UI/UX ด้วย Figma และจัดสไตล์ด้วย Tailwind CSS ฝั่ง Backend ใช้ Express.js บน Node.js เชื่อมต่อฐานข้อมูล MySQL พร้อมระบบ API ที่แยกเลเยอร์ชัดเจน รองรับฟังก์ชันหลัก ได้แก่ CRUD, Authentication และการจัดการสิทธิ์ผู้ใช้ด้วย RBAC

(EN) A full-featured E-Commerce web application built with React.js and Next.js on the frontend, styled with Tailwind CSS and designed in Figma. The backend is powered by Express.js (Node.js) with a well-structured layered API connected to a MySQL database. Core features include CRUD operations, Authentication, and Role-Based Access Control (RBAC).
